# KLINK

**Note**  
`Pastikan PC sudah terinstall redis, dan sudah running.
`   

## Penggunaan

### Persiapan Database
Buat database di mysql dengan nama klink

### Clone repository, dan install node packages

``` 
git clone https://github.com/dahaaw/KLINK.git
cd KLINK
npm install
npm start
```

### Testing
